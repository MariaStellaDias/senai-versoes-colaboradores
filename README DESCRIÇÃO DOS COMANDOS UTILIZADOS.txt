DESCRIÇÃO DOS COMANDOS UTILIZADOS:


1. git init: Cria a estrutura inicial do repositório Git no computador local.

2. git status: Verifica o status das alterações realizadas no repositório.

3. git add: Adiciona arquivos ao histórico do projeto, na staging.

4. git commit:Registra/salva a alteração no repositório.

5. git log: Permite visualizar as alterações feitas.

6. git show número-do-commit: Permite visualizar informações sobre qualquer commit.

7. git remote add otigin "destino": Informa a pasta remota. 

8. git remote -v: Permite visualizar o repositório remoto.

9. git push -u origin master:Publica as alterações no repositório remoto.

10. git pull: Baixa as alterações no repositório remoto.
